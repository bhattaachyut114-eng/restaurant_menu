# Restaurant Menu Ordering System

A simple Python program that displays a restaurant menu and allows customers to place orders interactively.

## Features

- **Menu Display**: Categorized menu with appetizers, main courses, desserts, and drinks
- **Order Taking**: Interactive ordering system with quantity selection
- **Order Summary**: Displays final order with totals
- **Error Handling**: Validates menu items and quantities

## How to Run

1. Make sure you have Python installed (version 3.6+)
2. Run the program:
   ```bash
   python restro_menu.py
   ```
3. Follow the prompts to place your order
